## 📝 API 사용 가이드라인

### 요청 형식
```http
POST /api/v1/endpoint
Content-Type: application/json
Authorization: Bearer {token}

{
  "data": {
    // 요청 데이터
  }
}
```

### 응답 형식
```json
{
  "success": true,
  "data": {
    // 응답 데이터
  },
  "meta": {
    "total": 0,
    "page": 1,
    "limit": 20
  }
}
```

### 에러 응답
```json
{
  "success": false,
  "error": {
    "code": "",
    "message": "",
    "details": {}
  }
}
```
